Lista de exemplos de HTML

HTML Básico

Todos os documentos HTML devem começar com uma declaração de tipo de documento: <!DOCTYPE html>.
O próprio documento HTML começa <html> e termina com </html>. A parte visível do documento HTML está entre <body> e </body>.

A <!DOCTYPE> declaração representa o tipo de documento e ajuda os navegadores a exibir as páginas da web corretamente.
Deve aparecer apenas uma vez, no topo da página (antes de qualquer tag HTML).

HTML Básico

Os títulos HTML são definidos com as tags <h1> to <h6>. <h1> define o título mais importante. <h6> define o título menos importante

Exemplo

Este texto é em <h1>

Este texto é em <h2>

Este texto é em <h3>

E assim por diante
